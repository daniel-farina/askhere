# AskHere

Desktop GUI for the [Grok CLI](https://x.ai/cli) — manage projects and conversations without living in the terminal.

This public repository is the **product monorepo home** for:
- 🌐 Marketing site (GitHub Pages)
- 📦 Binary releases (DMG today; more platforms later)

Application **source code is private** (`askhere-src`).
